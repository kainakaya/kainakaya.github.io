#website

BOOKexchangeSTORE

読む・買う・交換ができる本屋をコンセプトに架空の店舗サイトを制作しました。
お店の特徴を理解していただくために難しい言葉を使わずに読み手のことを考えて設計しました。

サイトを見る
ターゲット ご近所にお住まいの方々。隣接しているカフェに来る方。
目的 読む・買う・交換というコンセプトを分かりやすく伝える。
情報設計 スクロールした後すぐにお店のコンセプトが伝わるように設計しました。
各セクションについては、情報を最小限にすることで簡潔にまとめることを意識しました。
デザインプロセス コンセプトを推したいのでなるべくシンプルでわかりやすいWEBサイトを目指しました。
使用している色を同系統でまとめ、色数を少なくすることでよりシンプルに表現しています。
制作期間

企画、情報設計、WF / 3時間
デザイン / 3時間
コーディング / 6時間
使用ツール VS Code / Illustrator / Photoshop
